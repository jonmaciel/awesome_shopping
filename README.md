
## Install

    $ cd awesome_shopping
    $ bundle install
    $ rails g react_on_rails:install
    $ bundle && npm install


## Start & watch

    $ foreman start -f Procfile.dev

## On brwoser

    $ Acess your localhost (http://localhost:3000/)
    $ If you wanna check GraphQL stuff use `/graphiql` route (http://localhost:3000/graphiql)
